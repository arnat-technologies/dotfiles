# Arnat dotfiles

## Installation

```shell
source <(curl -s https://raw.githubusercontent.com/arnat-technologies/dotfiles/prd/remote-install.sh?$(date +%s))
```

## Symlinking files

```shell
. $DOTFILES/scripts/symlink.sh
```

## Features

- Symlink only files from `/files` folder
